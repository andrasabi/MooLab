# MooLab – Ferma Digitală 🐄

**Platformă educațională gamificată pentru sănătatea, bunăstarea și managementul digital al vacilor de lapte.**

MooLab este un joc educațional interactiv (o singură pagină HTML, fără instalare) destinat studenților, elevilor, fermierilor și specialiștilor din sectorul zootehnic. Utilizatorii își testează cunoștințele prin quiz-uri și rezolvă cazuri clinice reale din fermele de vaci de lapte, alegându-și rolul profesional și modul de joc.

Resursa a fost dezvoltată în cadrul proiectului **„Transformarea digitală a fermelor mici și mijlocii de vaci de lapte: o abordare integrată pentru îmbunătățirea productivității, eficienței și rezilienței”**, finanțat de **Academia Oamenilor de Știință din România (AOSR)**.

---

## ▶️ Cum se rulează

Nu necesită instalare sau server. Descărcați fișierul `index.html` (sau `MooLab_v2.0.html`) și deschideți-l în orice browser modern (Chrome, Edge, Firefox, Safari) pe desktop sau telefon. Pentru acces online, fișierul poate fi publicat prin GitHub Pages.

## 🎮 Conținut

**Moduri de joc:** Quiz rapid · Scenarii clinice · Complet (Quiz + Scenarii + Informații + Teste de laborator).

**Roluri profesionale (6):**
- Medic veterinar
- Inginer zootehnist
- Fermier
- Tehnician calitatea laptelui
- Specialist PLF (zootehnie de precizie)
- **Operator însămânțări artificiale** *(nou în v2.0)*

**Acoperire tematică:** stres termic (indicele THI), mastită și calitatea laptelui (NCS), boli metabolice periparturiente (cetoză, hipocalcemie), bunăstarea animală (cele 5 libertăți), zootehnie de precizie (senzori PLF), reproducție și **însămânțare artificială cu material seminal congelat**, sănătatea onglonului, program de înțărcare (dry-off), bariere în adoptarea tehnologiilor digitale și **analiză economică / surse de finanțare**.

- **15 întrebări de quiz** cu indicii, feedback și explicații detaliate.
- **11 scenarii clinice** cu date de monitorizare și consecințe ale deciziilor.
- **Modul de informații** cu ghidul practic THI, analiza economică (cost-beneficiu, finanțare DR-14 / SM 4.1 / PNDR) și barierele identificate în cercetarea AOSR.
- **Fișe de teste de laborator** (CMT, BCS, profil metabolic, NCS, BHBA, DCAD etc.).

## 🆕 Noutăți în versiunea 2.0

- Buton **Acasă** accesibil din orice ecran (siglă din bara de sus clicabilă) – rezolvă blocajele de navigare semnalate.
- Butoane **Anterior / Următor** dezactivate vizual la capete; navigare îmbunătățită în Scenarii și Teste.
- Layout **mobil** corectat: punctele cazurilor apar sub informații, urmate de butoane.
- **Casete pătrate** pentru selecția rolurilor; siglă mai mare în joc; fundal rural transparent pe scenarii; pictogramă nouă pentru rolul de Fermier.
- Corecții de terminologie: **onglon** (în loc de copită), **înțărcare / dry-off** (în loc de uscare).
- Conținut nou: rol **Operator însămânțări artificiale**, informații despre **material seminal congelat**, **modul de analiză economică**, **bariere în adoptarea tehnologiilor digitale** și **ghidul practic THI** al autorului.
